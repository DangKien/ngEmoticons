Features

Font Smiley Support Use the font smileys by using familiar codes like :) :/ :( etc. You can easily change the color of those smileys.
Emojis' Support They were everywhere so here they come. The most widely used emoji set (enhanced for retina display)
Code syntax highlighting Get inline code syntax highlighting using the code's markdown syntax with customizable themes.
Youtube/vimeo embedding Your simple youtube/vimeo links are automatically converted into embedded video with all the video details.
Text to HTML links Links from the text are converted into html anchor tags automatically.
PDF viewing The pdf viewing frame loads just after the text (if pdf link present) and makes it easy to view w/o downloading.
Multimedia File (Video+Audio+Images) embed All html5 supported multimedia files automaticall load in their respective media players.See examples.
Fully Customizable You have full control on everything.
